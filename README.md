## likelion-vite-react Setting Guide💡

### 저장소 코드를 복사합니다😁

```bash
npx degit dnqls9875/likelion-vite-react
```

### 설치순서 ✅

1️⃣ vscode에서 과제 폴더를 오픈합니다.

2️⃣ 터미널 창에서 위의 코드를 붙여넣기 합니다.

3️⃣ 패키지를 설치합니다.

```bash
pnpm i
```

4️⃣ 서버실행 및 컴파일을 실행합니다.

```bash
pnpm dev
```

### 실행 명령어 ✅
